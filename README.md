# Тест на дальтонизм | Colorblindness Test

Профессиональная платформа для диагностики нарушений цветового зрения.

## Особенности

✅ 4 типа клинически валидных тестов:
- Тест Ишихары (Ishihara)
- Тест Фарнсворта D-15 (Farnsworth)
- FM 100 Hue Test
- Аномалоскоп (Anomaloscope)

✅ Полностью анонимно и бесплатно
✅ Адаптивный интерфейс (мобильные + десктоп)
✅ Сохранение результатов
✅ История тестирования

## Технологии

- **Framework:** Next.js 14
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Database:** Prisma + SQLite
- **Runtime:** Node.js 18+

## Установка

```bash
npm install
```

## Развитие

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Сборка

```bash
npm run build
npm start
```

## Важно

⚠️ Этот тест является скрининговым и не заменяет профессиональную офтальмологическую консультацию.
